# 五色ダイエット30日間サポート付きプログラム LP

**デモサイト**

このLPは、五色ダイエット協会が提供する「30日間サポート付きプログラム」のランディングページです。

---

## 📋 概要

- **ターゲット：** 40〜60代女性
- **プログラム内容：** 
  - 個別Zoom相談（30分 × 3回）
  - 毎日の食事LINEサポート（14日間）
  - LINE・メール質問し放題（30日間）
- **価格：** 39,800円（48時間限定・通常49,800円）

---

## 🎨 デザイン仕様

- **カラー：** ピンクをベースに5色展開
- **フォント：** モバイル17px / PC18px
- **レイアウト：** モバイルファースト（80%がスマホ閲覧）
